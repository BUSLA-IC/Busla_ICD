Project Technical Stack & Development Rules

Before implementing any requested changes, follow these project constraints:

Tech Stack
Frontend: Vanilla JavaScript (ES6+), HTML, and Tailwind CSS.
Database: Supabase.
Architecture
This project does not use any frontend framework such as React, Vue, Angular, or similar.
This project does not use a traditional backend server (Node.js, Express, Laravel, Django, etc.).
Backend Logic

All backend functionality must rely only on Supabase features, including:

Supabase Database.
Supabase Functions.
Database Triggers.
Supabase Realtime.
Any other native Supabase capabilities when needed.

Do not introduce any external frameworks or backend technologies. All implementations must be compatible with the existing Vanilla JavaScript + HTML + Tailwind CSS + Supabase architecture.
